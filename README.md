# ArcGIS-Pro-Polygon-Based-Raster-Clipping-and-Spectral-Band-Value-Extraction.
This script automates polygon-based clipping and spectral value extraction from a multiband raster in ArcGIS Pro. For each polygon mask, the script clips the raster, separates the B, G, R, Red Edge, and NIR bands, converts the B band to points, and extracts all band values to the point attributes.
