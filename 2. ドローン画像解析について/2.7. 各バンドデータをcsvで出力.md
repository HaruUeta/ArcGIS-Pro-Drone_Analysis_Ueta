## 7. 最終的に、調査区内の各地点における5バンドの値を属性情報として持つポイントデータを作成し、csvファイルとして出力

ここでは、5バンドの値を属性情報として持つポイントデータをCSVファイルとして出力する。
<br>
<br>

### 入力データ

この処理で使用する入力データは、前の手順で作成した5バンドの値を持つポイントデータである。

例：

```text
Site7_9_points
```

このポイントデータには、以下のような属性情報が含まれている。

```text
B
G
R
REDEDGE
NIR
```
<br>
<br>

### 使用するツール

CSVとして出力するには、以下のツールを使用する。

```text
Conversion Tools
→ Excel
→ Table To Excel
```

または、

```text
Conversion Tools
→ To Geodatabase / To Table
→ Table To Table
```

CSV形式で出力したい場合は、Python上で属性テーブルをCSVとして書き出す方法も使用できる。
<br>
<br>

### 出力されるCSVファイル

最終的に、以下のようなCSVファイルを作成する。

```text
Site7_9_points.csv
```

CSVファイルには、各ポイントの位置情報と、5バンドの反射値が記録される。

出力される列の例：

```text
OBJECTID
POINT_X
POINT_Y
B
G
R
REDEDGE
NIR
```
