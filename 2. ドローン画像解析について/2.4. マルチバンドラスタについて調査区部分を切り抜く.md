## 4. 作成したマルチバンドラスタデータを調査区部分のみを切り抜く

目視にて、設置された標識を見つけ、マルチバンドラスタデータについて調査区部分のみを切り抜くためのポリゴンを作成する。
<br>

「プロジェクト」→「データベース」→「XXX.gbb（動かしているプロジェクトデータ）」を右クリック。
「新規」→「フィーチャクラス」で名前をつけてポリゴンを作成する。
<br>

この際、50 cm × 50 cmのプロットを4等分したサブプロット単位でポリゴンを作成した。
サブプロットは、北を上にした時の左上→右上→左下→右下の順で1→2→3→4と番号をつけた。

<img width="2346" height="2177" alt="図2" src="https://github.com/user-attachments/assets/6a516ac4-f31b-4ea8-920e-26b35aab2680" />


ポリゴンを全プロット数×4つ分作成するのがめちゃくちゃ手間なので、以下のコードでまとめて作成するのが良いと思う。
ArcGISの「解析」のPythonから以下のマップ名などを適宜変えて実行すればいけるはず。
<br>

※たとえば a_1 と入力したら、以下の 8個の空ポリゴンレイヤーをまとめて作るコードなので注意。

```bash
a_1_1_1
a_1_1_2
a_1_1_3
a_1_1_4
a_1_8_1
a_1_8_2
a_1_8_3
a_1_8_4
```

以下が本コードである。

```bash
# -*- coding: utf-8 -*-

import arcpy
import os
import re


# 0. 設定

# 対象マップ名
map_name = "Site7_9"

# 入力する名前
# 例: "a_1", "b_3", "Site_2" など
input_name = "a_1"

# 出力先GDB
# 現在のArcGIS ProプロジェクトのデフォルトGDBを使う
aprx = arcpy.mp.ArcGISProject("CURRENT")
output_gdb = aprx.defaultGeodatabase

# 座標系
# None の場合、マップの座標系を使用
spatial_reference = None

# 作成したポリゴンをマップに追加するか
add_to_map = True


# 1. 関数

def safe_name(name):
    """
    ArcGISで使える名前に整える
    """
    name = re.sub(r"[^\w]", "_", name)
    name = re.sub(r"_+", "_", name)
    name = name.strip("_")

    if re.match(r"^\d", name):
        name = "n_" + name

    return name


# 2. マップ取得

maps = aprx.listMaps(map_name)

if not maps:
    print("現在のプロジェクト内のマップ一覧:")
    for m in aprx.listMaps():
        print(f"  - {m.name}")
    raise ValueError(f"マップ '{map_name}' が見つかりません。")

m = maps[0]

if spatial_reference is None:
    spatial_reference = m.spatialReference

print(f"対象マップ: {m.name}")
print(f"出力GDB: {output_gdb}")
print(f"座標系: {spatial_reference.name}")


# 3. 入力名を分解

# 例: a_1 → alphabet = a, number = 1
parts = input_name.split("_")

if len(parts) != 2:
    raise ValueError(
        "input_name は 'a_1' のように、"
        "アルファベット_番号 の形で入力してください。"
    )

alphabet = parts[0]
number = parts[1]

alphabet = safe_name(alphabet)
number = safe_name(number)

print(f"\n入力:")
print(f"  アルファベット: {alphabet}")
print(f"  番号: {number}")


# 4. 作成するポリゴン名リスト

polygon_names = []

for main_num in [1, 8]:
    for sub_num in [1, 2, 3, 4]:
        name = f"{alphabet}_{main_num}_{number}_{sub_num}"
        polygon_names.append(safe_name(name))

print("\n作成するポリゴン:")
for name in polygon_names:
    print(f"  - {name}")



# 5. ポリゴンフィーチャクラス作成

created_fcs = []

for poly_name in polygon_names:

    out_fc = os.path.join(output_gdb, poly_name)

    if arcpy.Exists(out_fc):
        print(f"既存のためスキップ: {poly_name}")
        created_fcs.append(out_fc)
        continue

    print(f"作成中: {poly_name}")

    arcpy.management.CreateFeatureclass(
        out_path=output_gdb,
        out_name=poly_name,
        geometry_type="POLYGON",
        spatial_reference=spatial_reference
    )

    created_fcs.append(out_fc)

    print(f"  完了: {out_fc}")


# 6. マップに追加

if add_to_map:
    print("\nマップに追加中...")

    for fc in created_fcs:
        try:
            m.addDataFromPath(fc)
            print(f"  追加: {os.path.basename(fc)}")
        except Exception as e:
            print(f"  追加できませんでした: {fc}")
            print(e)

print("\nすべて完了しました。")
print("作成されたポリゴンフィーチャクラス:")

for fc in created_fcs:
    print(fc)
```

<br>
※この際、調査区の座標データcsvファイルを取り込んで、マップ上にプロットするとわかりやすい。

### 調査区座標CSVの読み込み

調査区ポリゴンを作成する前に、調査区の座標データを含むCSVファイルをArcGIS Proに読み込み、マップ上にポイントとして表示する。これにより、各調査区の位置を確認しながらポリゴンを作成できる。

流れは以下の通り。

```bash
1. 対象のマップを表示する。

2. 「マップ」タブから「データの追加」を選択し、調査区の座標データを含むCSVファイルを読み込む。

3. 読み込んだCSVテーブルを右クリックし、「XYテーブル → ポイント」を選択する。

4. Xフィールドに経度またはX座標、Yフィールドに緯度またはY座標を指定する。

5. 座標系を指定する。緯度・経度データの場合は、通常 WGS 1984 を指定する。平面直角座標系などを用いている場合は、データに対応した座標系を指定する。

6. 実行
```

一旦作るとコピー貼り付けでいろんなマップに貼り付けられる。
